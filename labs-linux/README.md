# Linux Labs 
Esta seccion contiene laboratorios practicos realizados sobre sistemas Linux. 

El objetivo de estos labs no es solo explotar sistemas, sino **observar, entender y documentar** su comportamiento desde una perspectiva defensiva pero no de manera exclusiva.

## Enfoque 
- Análisis del estado del sistema.
- Observacion de procesos y servicios
- Revision de actividad de red.
- Identificacion de comportamientos normales y anomalos

## Metodología 
Cada laboratorio sigue una estructura clara:
- Definir un objetivo
- Ejecutar acciones controladas
- Registrar observaciones
- Extraer conclusiones

## Nota
Los comandos y herramientas utilizados se documentan con contexto. 
El enfasis esta en el razonamiento y no en la automatizacion ciega.
