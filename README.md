# Cybersecurity-Lab-Knowledge-Base
Documentación práctica, análisis y experimentación orientada a defensa, observación y comprensión real de sistemas.

Este repositorio documenta mi proceso se aprendizaje practico en ciberseguridad, con enfoque en observación, análisis y defensa de sistemas de Linux y Windows.

## Metodología
- Observar
- Entender
- Ejecutar
- Documentar

## Contenido
- Fundamentos de sistemas y redes
- Laboratorios prácticos documentados
- Análisis de procesos, red y eventos
- Scripts simples para automatización

## Objetivo
Construir una base solida en Blue team, comprender como funciona realmente sobre el uso de herramientas automaticas.
